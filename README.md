# outdoor_clubs

戶外活動社網站

## 初始化

安裝

```
npm install -g @vue/cli
```

建立專案

```
vue create outdoor_clubs
```

## Vue Cli 設定

```
npm install
```

### 為開發編譯和熱重載

```
npm run serve
```

### 編譯並縮小以用於生產

```
npm run build
```

### Lints 和修復文件

```
npm run lint
```

### 自定義配置

請詳閱[config 設定參考](https://cli.vuejs.org/config/)。

## 安裝紀錄

### sass loader

安裝最新版的會失敗, 解法如下:

```
npm install sass-loader@10 node-sass --save-dev
```
